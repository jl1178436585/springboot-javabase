1.二叉树
	1.1 每个节点最多有两个子树的树结构 
	1.2  二叉树的每一个节点不一定是满的，也就是说不一定每个节点都有两个孩子

2.二分搜索树是一个二叉树

3.二分搜索树（Binary Search Tree）BST的每个节点的值：
	大于其左子树的所有节点的值
	小于其右子树的所有节点的值
   所以	存储的元素必须是具有可比较性	
   3.1 二分搜索树的每一棵子树也是二分搜索树

   3.2 深度优先遍历
   		3.2.1 前序遍历
   		3.2.2 中序遍历
   		3.2.3 后序遍历 	
   		
   3.3 广度优先遍历:由于查找元素快，跟多用于搜索
		3.3.1 层序遍历	   
     

4.满二叉树： 
	4.1定义一： 一棵深度为k，且有2^k-1个节点的树是满二叉树。
	4.2另一种定义：除了叶子结点外每一个结点都有左右孩子且叶子结点都处在最底层的二叉树。所有叶子结点必须在同一层上
	
5.完全二叉树：
    5.1 若设二叉树的深度为h，除第 h 层外，其它各层 (1～h-1) 的结点数都达到最大个数，第 h 层所有的结点都连续集中在最左边，这就是完全二叉树。
	5.2  叶子结点最大的深度值和最小的深度值相差不会超过1，就是说叶子节点要么在最底层要么在倒数第二层。
	5.3  除最后一层外，每一层上的节点数均达到最大值；在最后一层上只缺少右边的若干结点

6.AVL：是最先发明的自平衡二叉树，也被称为高度平衡树
	4.1 G.M.Adelson-VelSky和E.M.Landis名字的简写
		 是最早的自平衡二分搜索树结构
		满二叉树一定是一个平衡二叉树
	4.2 对于任意一个节点，左子树和右子树的高度差不能超过1	
	4.3 平衡二叉树中叶子结点的高度始终是1

7.树的高度和深度：
	7.1 高度：楼房的高度从下往上数，所以树的高度也是从下往上。
	7.1 深度：是从根节点往下
	
8.二三树：
	a.满足二分搜索树的基本性质
	b.节点可以存放一个元素或者两个元素
	c.每一个节点有2个或者3个孩子，所以不是二叉树
	d.任意一个节点的左右子树的高度是相等的

	2-3树是一颗绝对平衡的树
	绝对平衡：从任意一个节点到任意一个叶子节点所经过的节点数量一定是相同的	
9.红黑树：
	9.1
		如果二三树中每一个节点只存储一个元素，那就是红黑树
		红黑树是一个保持黑平衡的二叉树，严格意义上不是平衡二叉树
	9.2 特点：
		1).没个节点要么是红色的要么是黑色的
		2).根节点是黑色的
		3).每一个叶子结点(最后的空节点，不是左右子树都为空的节点)
		4).如果一个节点是红色的，那么他的孩子节点都是黑色的
		5).从任意一个节点到叶子结点，经过的黑色节点是一样的
	9.3	红黑树和平衡二叉树AVL的对比
		1).查询效率AVL要好一点，但是差距不大
		2).增加和删除效率红黑树好，红黑树最大高度2(logn) ---> O(logn)
